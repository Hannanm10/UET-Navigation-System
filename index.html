UET Campus Navigation System

Welcome to UET Campus Navigation System

The Campus Navigation System simplifies navigation within a university by providing an interactive tool to select current locations and destinations, displaying optimal routes on a map. It also logs route history with timestamps for user convenience and administrative insights, enhancing campus accessibility and efficiency.

About the Project

The Campus Navigation System enhances user experience by providing efficient route planning and maintaining a detailed route history, ensuring better campus navigation and accessibility.

The proposed solution uses a user-friendly interface to provide real-time route guidance, log travel history by capturing selected destinations and timestamps, and enable efficient navigation across campus locations.
